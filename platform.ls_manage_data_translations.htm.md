# Manage Objects’ Data Translations

[](https://help.salesforce.com/s?language=en_US)You can manage data translation for Work Type Group, Service Territory, and Service Resource objects. Use data translation to build Scheduler’s appointment booking experience in multiple languages. For example, you can translate the work type group names and their description to Spanish for a customer who uses the website in Spanish to schedule appointments.

-   **[Considerations for Data Translation of Objects](https://help.salesforce.com/s/articleView?id=platform.ls_considerations_data_translations.htm&language=en_US&type=5)**  
    Here are some things to keep in mind when you translate object data.
-   **[Enable Data Translation](https://help.salesforce.com/s/articleView?id=platform.ls_enable_data_translation.htm&language=en_US&type=5)**  
    Enable users to translate the data for Work Type Group, Service Territory, and Service Resource objects.
-   **[Enable Translation Workbench](https://help.salesforce.com/s/articleView?id=platform.ls_enable_translation_workbench.htm&language=en_US&type=5)**  
    Use Translation Workbench to specify languages for translation, to assign translators, and to manage your translations.
-   **[Add Translation Languages and Translators](https://help.salesforce.com/s/articleView?id=platform.ls_add_translation_language_and_translators.htm&language=en_US&type=5)**  
    Add languages for translation, assign translators for each language, and activate or deactivate a language’s translations.
-   **[Enable Translation of Custom Field Data](https://help.salesforce.com/s/articleView?id=platform.ls_enable_translation_custom_fields.htm&language=en_US&type=5)**  
    Enable translation of custom field data for Work Type Group, Service Territory, and Service Resource objects. You can enable data translation for custom fields with Text, Text Area, Text Area (Long), Text Area (Rich), and URL types.
-   **[Add the Translations Tab to an Object](https://help.salesforce.com/s/articleView?id=platform.ls_add_translations_tab.htm&language=en_US&type=5)**  
    To associate translations with their objects, add the Translations tab.
-   **[Add Translations to Standard and Custom Fields](https://help.salesforce.com/s/articleView?id=platform.ls_add_translations_std_and_custom_fields.htm&language=en_US&type=5)**  
    Add translations to standard and custom fields of Work Type Group, Service Territory, and Service Resource objects.
-   **[Import and Export Translations](https://help.salesforce.com/s/articleView?id=platform.ls_import_and_export_translations.htm&language=en_US&type=5)**  
    You can import and export translated data files.
