# Add Translations to Standard and Custom Fields

Add translations to standard and custom fields of Work Type Group, Service Territory, and Service Resource objects.

Before adding data translations, ensure that you add the Translations tab to the object.

1.  Click an object tab, and then select an instance of the object.
    
    For example, click the **Service Territories** tab, and then select a service territory.
    
2.  Click the **Translations** tab, and then click **Add Translation**.
3.  Select a language.
4.  Enter the translated text for the name, description, and custom fields.
5.  Save your changes.
