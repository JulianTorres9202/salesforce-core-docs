# Manually Localize Store Labels

Some B2B component labels are translated for you into the store's selected language, but you need to manually translate some labels. To translate your store’s text elements in bulk, use the translation tool.

1.  On your store’s home page, click **Website Design**, and then click **Experience Builder**.
2.  At the top of the navigation bar, click [](https://help.salesforce.com/s?language=en_US) and select the store’s language.
3.  Enter the localized text in the component property fields.
