# Enable Translation of Custom Field Data

Enable translation of custom field data for Work Type Group, Service Territory, and Service Resource objects. You can enable data translation for custom fields with Text, Text Area, Text Area (Long), Text Area (Rich), and URL types.

Before enabling data translation for custom fields, ensure that you enable data translation.

1.  From Setup, in the Quick Find box, enter Data Translation Settings, and then select **Data Translation Settings**.
2.  To enable data translation for custom fields, select an object.
    
    Only objects that support data translation are listed.
    
3.  Select the custom fields for which you want to enable translation.
4.  Save your changes.
