# Copy Multilanguage artifacts

Multilanguage support allows users to import app configs, email templates, layouts, and reason codes in different languages, enhancing global brand consistency. Other artifacts default to the primary language, ensuring seamless integration.

Important It isn't recommended to use this feature between a merchant account configured for one language and one configured for multilanguage. Such use may cause bugs and unintended side effects.

[](https://help.salesforce.com/s?language=en_US)

## Import a bundle in a different language

1.  In CMS, go to **Advanced** | **Copy Tools** | **Import**.
2.  Search for the exported bundle using the filters.
3.  Click **Search**.
4.  Select the bundle(s) you want to import.
5.  Select the language you want to import the artifacts in using the language picker.
6.  Click **Import**.

[](https://help.salesforce.com/s?language=en_US)

## View artifact(s) in a specified language

1.  In CMS, go to **Advanced** | **Copy Tools** | **Export**.
2.  Under Artifacts, click + Add and select a resource.
3.  Toggle the languages with the language picker. Selecting a language displays artifacts that are set in that language. For example, selecting French (FR) displays artifacts in the French language.
