# Data Translation

When data translation is enabled, the data stored in certain objects and fields are available for translation. You can also enable data translation for custom text and URL fields on objects.

Note Data translation requires API version 48.0 or later.

-   **[Enable Data Translation](https://help.salesforce.com/s/articleView?id=platform.data_translation_enable.htm&language=en_US&type=5)**  
    Allow translation of data stored in certain objects and fields.
-   **[Enable Data Translation for Custom Fields](https://help.salesforce.com/s/articleView?id=platform.data_translation_enable_custom_fields.htm&language=en_US&type=5)**  
    Allow translation of data stored in custom fields on certain objects and fields. You can enable data translation on custom fields with a type of Text, Text Area, Text Area (Long), Text Area (Rich), and URL.
