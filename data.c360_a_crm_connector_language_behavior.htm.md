# CRM Connector Language Behavior

Keep these considerations in mind when when using the CRM connector.

The Platform Integration User's language is updated to the new language when the default organization language is changed.

The default language set at the organization level determines the results of a knowledge article query.

Source field names default to the language used by the object. When adding source fields during data stream creation, ensure that the default language matches the object's language. If the languages don't match, the fields won't be available for selection.
