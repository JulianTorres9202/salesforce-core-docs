# Languages Localization on the POS App

Set up language localization deep links in the POS app through Point of Sale CMS configurations.

Different languages can also be set in the POS app. This is a two-step process involving initial configurations via Point of Sale CMS and then the app. In Point of Sale CMS, the deep link has to be set to change the language in the app.

1.  Go to **CMS > Apps > All > your app config > Edit**.
2.  Select **Edit Layout**.
3.  Select where you want to assign a deep link to it. For example in a cell, in the hamburger menu, and so on.
4.  In the right toolbar, select **Deep link**.
5.  Do a search for "Language Picker".
6.  Select **Language Picker** and select **SET**.
7.  Select **PUBLISH NOW** or **Schedule**.
