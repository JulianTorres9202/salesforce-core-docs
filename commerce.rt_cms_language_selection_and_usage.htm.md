# Language Selection and Usage

The POS app lets users change the language at any time or select a language for a specific operation, such as searching for products or printing receipts.

## The POS app has functionalities that the store uses to change the language at any time or select a language for a specific operation. The following aspects of the app are considered:

1.  Selecting a display language on the app
2.  Searching for products
3.  Print receipts
4.  Offline functions

## Selecting a Display Language on the App

1.  Set a deep link via CMS. For instructions on how to set up a deep link, see [Languages Localization on the POS App](https://help.salesforce.com/s/articleView?id=commerce.rt_cms_localization_pos_app.htm&language=en_US&type=5 "Set up language localization deep links in the POS app through Point of Sale CMS configurations.").
2.  Restart the app and log in.
3.  In the app, select the language deep link you've set.
4.  Select the language and select Apply.

After a successful language update, the app will fetch the language-specific app configuration and show the updated layouts and labels.

## Resetting the language

In the language picker screen, you can always reset the previously set language and fetch the app configuration and other settings based on the default language set in the iPad settings.

## Searching for Products

After the language is set on the app, the products via a product search now show the specific text and result in the language selected.

## Printing Receipts

The app will show the supported languages on the Thank You screen, allowing you to print the order receipts in a specific language.

The Thank You screen is a layout that can be changed through Point of Sale CMS. The language picker can be set up as a form element to print receipts in a different language.

For more information on layouts and form elements, see Layout - Cell Configuration
