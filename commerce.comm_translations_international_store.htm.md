# Manage Translations for B2B Commerce Stores

You can translate the product, category, and promotion text. If you’re using product variations, you can also translate the variation attribute names. In Experience Builder, you can localize the store component labels.

[Which Salesforce Commerce Product Do I Have?](https://help.salesforce.com/s/articleView?id=commerce.comm_what_product.htm&language=en_US&type=5)

-   **[Enable Data Translation](https://help.salesforce.com/s/articleView?id=commerce.comm_enable_data_translation.htm&language=en_US&type=5)**  
    When you enable data translation, you can translate the data stored in your B2B store product, category, and promotion fields. You can also enable data translation for custom text and URL fields on the product and category objects.
-   **[Enable Translation Workbench](https://help.salesforce.com/s/articleView?id=commerce.comm_translation_workbench.htm&language=en_US&type=5)**  
    To assist with localizing B2B store information, enable the Translation Workbench and add the languages that you want to use in your stores.
-   **[Manually Localize Product, Category, and Promotion Data](https://help.salesforce.com/s/articleView?id=commerce.comm_create_data_translation.htm&language=en_US&type=5)**  
    You can manually enter translated text for your B2B product and category names and descriptions. To translate promotion names and descriptions, you must use Translation Workbench. To perform bulk translations for product, category, or promotion data, use Translation Workbench.
-   **[Localize Product Variation Data](https://help.salesforce.com/s/articleView?id=commerce.comm_var_att_localization.htm&language=en_US&type=5)**  
    If you’re using product variations in your B2B store, you can translate the variation attribute names into languages that your org supports to offer product variations in multiple locales.
-   **[Manually Localize Store Labels](https://help.salesforce.com/s/articleView?id=commerce.comm_translate_store_labels_manually.htm&language=en_US&type=5)**  
    Some B2B component labels are translated for you into the store's selected language, but you need to manually translate some labels. To translate your store’s text elements in bulk, use the translation tool.
-   **[Localize Store Labels in Bulk](https://help.salesforce.com/s/articleView?id=commerce.comm_translate_store_labels_translation.htm&language=en_US&type=5)**  
    When you add languages to Experience Builder, some component labels are translated for you into the selected language. If you add more languages, you can translate your B2B store’s text elements in bulk by using the translation tool. You can also translate configurable labels manually in the component properties.
-   **[Bulk Translation for a B2B Store](https://help.salesforce.com/s/articleView?id=commerce.comm_bulk_translate_b2b_store.htm&language=en_US&type=5)**  
    To accomplish bulk translations of B2B store metadata and Product, Category, and Promotion fields, use the Translation Workbench source export option. The Translation Workbench supports the XML Localization Interchange File Format (XLIFF) and the Salesforce Translation File (STF) file formats. Salesforce recommends that you use the XLIFF format.
