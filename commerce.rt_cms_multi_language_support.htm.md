# Multi-Language Support

POS supports multiple languages in the same country, which benefits merchants in different locales across the globe.

The POS can localize as part of its main platform features. This means you can run multiple languages in the same country. This can be helpful for merchants based in different locales across the globe, allowing languages to be switched within one account.

Note Multilanguage support is only available to merchants configured for it during their initial setup, a process managed by Salesforce. For more information, contact Salesforce Customer Support.

## Supported Languages

The modern POS app supports these languages out of the box:

-   English (default)
-   French
-   Polish
-   Spanish
-   Portuguese
-   Japanese
-   Dutch
-   Italian
-   German
-   Chinese, Simplified
-   Catalan
-   Malay

Note Point of Sale leverages machine translation to automatically translate UI text on the POS app.

## Prerequisites

-   Modern POS App Build: 7.16.5 & later.
-   Platform Support: iPhone & iPad
-   Offline Support: yes, but default language only

## Use Cases Supported

-   Configuring merchants with multiple languages in the same region.
-   Product feeds for importing into Point of Sale.
-   Multiple-language localization on the POS app.
-   Multiple language configurations on the CMS for:
    -   POS app configuration
    -   Email & Print templates
    -   Layouts
    -   Promotions
    -   Reason codes
    -   Tax
-   Language selection in the POS app user interface.

-   **[Configuring Merchants with Multiple Languages](https://help.salesforce.com/s/articleView?id=commerce.rt_cms_merchants_multiple_languages.htm&language=en_US&type=5)**  
    The Point of Sale platform lets you configure merchants based on MerchantID, Locale, and Region.
-   **[Product Feeds for Multiple Languages](https://help.salesforce.com/s/articleView?id=commerce.rt_cms_product_feeds.htm&language=en_US&type=5)**  
    Use product feeds in Point of Sale CMS to manage products in multiple languages.
-   **[Languages Localization on the POS App](https://help.salesforce.com/s/articleView?id=commerce.rt_cms_localization_pos_app.htm&language=en_US&type=5)**  
    Set up language localization deep links in the POS app through Point of Sale CMS configurations.
-   **[Multiple Language Configuration on Point of Sale CMS](https://help.salesforce.com/s/articleView?id=commerce.rt_cms_multiple_language_configuration.htm&language=en_US&type=5)**  
    Point of Sale CMS lets you set up POS apps in different languages. You can also set up email and print templates, layouts, promotions, reason codes, and tax.
-   **[Language Selection and Usage](https://help.salesforce.com/s/articleView?id=commerce.rt_cms_language_selection_and_usage.htm&language=en_US&type=5)**  
    The POS app lets users change the language at any time or select a language for a specific operation, such as searching for products or printing receipts.
-   **[Troubleshooting](https://help.salesforce.com/s/articleView?id=commerce.rt_cms_multilanguage_troubleshooting.htm&language=en_US&type=5)**  
    Make sure that your device's region and language settings are correctly configured to avoid errors while loading a specific app configuration in the Point of Sale CMS Point of Sale (POS) App.
