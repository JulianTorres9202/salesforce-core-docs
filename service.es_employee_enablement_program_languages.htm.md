# Supported Languages for Employee Enablement Program

Most features for creating and taking employee enablement programs are available in all languages that Salesforce supports. But some records, content, and other items support only one language.

-   Program titles and descriptions
-   Exercise titles and descriptions
-   Content that you add for exercises
